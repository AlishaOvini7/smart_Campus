API Online

Smart Campus API

Manage campus rooms and IoT sensors easily.
Use tools like Postman or curl to test endpoints.

Discovery

GET /api/v1/ API info

Rooms

GET /api/v1/rooms All rooms
POST /api/v1/rooms Create room
GET /api/v1/rooms/{id} Single room
DELETE /api/v1/rooms/{id} Remove room

Sensors

GET /api/v1/sensors All sensors
GET /api/v1/sensors?type=CO2 Filter by type
POST /api/v1/sensors Add sensor
GET /api/v1/sensors/{id} Single sensor
DELETE /api/v1/sensors/{id} Remove sensor

Readings

GET /api/v1/sensors/{id}/readings Sensor data
POST /api/v1/sensors/{id}/readings Add reading
